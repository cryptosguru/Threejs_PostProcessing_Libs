Defined in src\passes\pass.js:78

Updates this pass with the new main render size. This is an abstract method that may be overriden in case
you want to react to render size changes.

Parameters:

  • w Number
    • The on-screen render width.
  • h Number
    • The on-screen render height.

Example:

this.myRenderTarget.width = w / 2;
